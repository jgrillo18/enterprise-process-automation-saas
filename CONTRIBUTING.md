# Contributing

Gracias por tu interés en contribuir a este proyecto.

## Cómo colaborar

1. Haz un _fork_ del repositorio.
2. Crea una rama nueva (`git checkout -b feature/mi-nueva-funcion`).
3. Realiza tus cambios y añade tests/documentación cuando sea necesario.
4. Haz `commit` y `push` a tu fork.
5. Abre un _pull request_ describiendo tus modificaciones.

## Directrices

- Mantén el estilo de código consistente (PEP8 para Python).
- No incluyas credenciales ni datos sensibles.
- Añade documentación en el README si agregas nuevas características.
